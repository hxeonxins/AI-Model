# **인공지능 모델링 수행평가 프로젝트 안내** 

## **1. 수행평가 1 (30점)** 
### **📌 주제: 실생활 이미지 분류 AI 모델 개발** 
### **📌 프로젝트 요구사항** 
### **1) 데이터셋 선택 (5점)** 
다음 중 **하나의 주제**를 선택하세요. 

**제시된 주제:** 
- **음식 분류 AI**: 한식, 중식, 일식, 양식 등 음식 종류 분류 
- **교통수단 분류 AI**: 자동차, 버스, 자전거, 오토바이 등 분류 
- **감정 분류 AI**: 표정 이미지를 통한 감정(행복, 슬픔, 화남 등) 분류 
- **질병 진단 AI**: 피부 질환, 식물 병해 등의 이미지 분류 
- **재활용 분류 AI**: 플라스틱, 종이, 캔, 유리 등 재활용품 분류 

**데이터 수집 요구사항:** 
- 최소 **3개 이상의 카테고리** 
- 각 카테고리당 최소 **200장** 이상의 이미지 
- 데이터 출처: Kaggle, Google Images 등 

### **2) 모델 개발 (15점)** 
**필수 구현 사항:** 
- CNN 레이어 최소 2개 이상 포함 
- 적절한 활성 함수 사용 (relu, sigmoid, softmax 등) 
- Dropout 또는 다른 overfitting 방지 기법 적용 
- Image Augmentation 적용 (최소 2가지 이상) 
- 학습/검증 데이터 분리 (8:2 또는 7:3) 

### **3) 성능 최적화 (10점)** 
**최소 3가지 이상의 최적화 기법 적용:** 
- Hyperparameter 튜닝 (learning rate, batch size, epochs 등) 
- 전이학습 (InceptionV3, ResNet, VGG 등) 
- Fine tuning 
- 다양한 Image Augmentation 조합 
- Learning rate optimizer 변경 (Adam, RMSprop 등) 
- Early Stopping 적용 
- 모델 아키텍처 변경 및 비교 

**성능 목표:** 
- Validation Accuracy: 95**% 이상** 
- Overfitting 정도: Train/Val accuracy 차이 5**% 이내**
